字符串编辑距离

发表于 2018-06-08 | 分类于 Algorithm

摘要： 基本动态规划

给你两个字符串
例如： word1: horse, word2:ros
你有三种选择：

  1. 插入
  2. 删除
  3. 替换
    把一个字符串变为另一个字符串

我们可以构造一张二维表

 0 s t r 1
|s 0 1 2 3
|t 1
|r 2
|2 3
